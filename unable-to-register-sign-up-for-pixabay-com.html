1 解决办法

该网络问题是由该网站上某些文件无法访问而引起的，在找到这些问题文件之后，用网址重定向的方法，将问题文件指引到正常的相同的文件，在替换之后，访问的加载速度也会明显加快。

1.1 Firefox浏览器

在 Firefox 火狐浏览器 - 全新、安全、快速 | 官方最新下载: http://www.firefox.com.cn/ 处下载和安装浏览器。

1.2 Redirector扩展

在火狐浏览器里安装 Redirector – 下载 Firefox 扩展（zh-CN）: https://addons.mozilla.org/zh-CN/firefox/addon/redirector/?src=search 。

1.3 Create new redirect

找到 问题的源头 ， 再添加 重定向的规则 。

2 问题的源头

需要找到那些有问题的文件，在谷歌浏览器里刷新注册页面 https://pixabay.com/zh/accounts/register/ ，会发现在＂Console＂和＂Network＂里，有个网址无法访问，提示： 

GET https://www.google.com/recaptcha/api.js net::ERR_CONNECTION_CLOSED

定位到＂Sources＂里，找到问题的源头在 (index):113： 

<script src="https://www.google.com/recaptcha/api.js" async defer></script>

等到加载完成后，又有个网址无法访问：提示： 

GET https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js net::ERR_CONNECTION_RESET

同样的问题在 (index):105： 

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>

上述两个网址无法访问，直接在新的页面里打开是显示无法访问的，需要 替换问题网址 。

3 替换问题网址

搜索可以访问的镜像网站，一一对应。

3.1 /jquery/1.12.4/jquery.min.js

找中国境内＂jquery.min.js＂文件的镜像网站 jquery | BootCDN - Bootstrap 中文网开源项目免费 CDN 加速服务: https://www.bootcdn.cn/jquery/ 里有相同1.12.4版本的文件 https://cdn.bootcss.com/jquery/1.12.4/jquery.min.js ，确认可以直接访问。

3.2 /recaptcha/api.js

同样是找中国境内＂Google reCAPTCHA＂的镜像网站 中国的网站怎么调用Google 的API？ - 知乎: https://www.zhihu.com/question/35269619 ，在其中提到 http://recaptcha.net ，替换域名后得到 https://recaptcha.net/recaptcha/api.js ，确认可以直接访问。

4 重定向的规则

在火狐浏览器＂Redirector＂扩展的规则编辑页面 moz-extension://66ead244-a02f-416c-bf0e-d90eac7559fc/redirector.html 里用＂Create new redirect＂手动创建规则，需要注意的是，上述的问题网址都存在于＂script＂里，所以需要在＂Show advanced options…＂的＂Apply to＂里增加＂Scripts＂功能。

也可以在规则编辑页面用＂Import＂来导入已经创建好的规则。

折腾不易，伸手党可以使用已经写好了的一个规则，下单 Pixabay.com无法注册.json-淘宝网: https://item.taobao.com/item.htm?id=603922090521 之后会收到已经创建好的规则，另存为文件后再导入，刷新注册页面，完成注册。

5 推而广之

不少使用Google服务的网站都有与pixabay站点相同的无法登录、注册、回复问题，包括但不限于：

  • https://stackoverflow.com/ Stack Overflow requires external JavaScript from another domain, which is blocked or failed to load. 里script问题文件 https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js
  • https://stackexchange.com/
  • https://androidfilehost.com/ 里点击＂Click Here to Start Download＂而无法下载的问题文件 https://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js?v=5.1.9
  • https://userstyles.org/

这些网站的加载速度不仅慢，还有问题，但为什么不能直接使用他人创建好的重定向规则，譬如 Search · Redirector.json · GitHub: https://github.com/search?q=Redirector.json&type=Commits 里无数的重定向数据，原因是并不能保证重定向的文件里是否包含窃取用户隐私数据的程序，这些名义上相同的文件，是否经过了修改也是不得而知，而使用经过自己验证过的，确实安全、稳定、可靠的正常文件，从零开始，保留一份自用的上网指引规则，并且可以随时随地地修改。
